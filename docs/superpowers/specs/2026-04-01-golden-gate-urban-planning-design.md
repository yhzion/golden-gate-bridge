# Golden Gate Bridge Urban Planning Design Spec

## Overview

금문교를 중심축으로 한 자연 친화형 생태 관광도시 설계. 5개 분야 전문가 서브에이전트(정치, 경제, 사회, 과학, 기술)가 병렬 독립 분석을 수행하고, 통합 조정 에이전트가 최종 마스터플랜을 작성한다.

### 핵심 원칙

- **자연 친화형 생태도시**: 건물이 자연과 어우러지는 싱가포르 가든시티 스타일
- **시네마틱 관광 쇼케이스**: 모든 요소가 카메라 비행 시 영화처럼 보여야 함
- **금문교 중심축 방사형 배치**: 모든 도로, 구역, 시설이 금문교를 향해 방사형으로 배치
- **해변 적극 활용**: 해안선을 관광 자원으로 활용
- **샌프란시스코와 무관**: 독립된 신도시 세계관

### 목표 시설

개척, 도로시스템, 주거환경, 상업시설, 편의시설, 휴양시설

### 산출물

설계 문서만 생성 (3D 코드 구현은 별도 단계)

### 실행 방식

- 5개 전문가 서브에이전트를 `Agent` 도구로 병렬 실행 (단일 메시지에서 5개 동시 호출)
- 각 에이전트는 독립적으로 분석하고 보고서를 파일로 작성
- 통합 조정 에이전트가 5개 보고서를 읽고 최종 마스터플랜 작성

## 도시 구역 배치 — 방사형 마스터플랜

### 내부 링 (금문교 반경 ~500m) — 관광·자연 중심

차량 진입 제한, 보행자 우선 구역.

| 구역 | 시설 | 성격 |
|------|------|------|
| 관광·문화 구역 | 전망대, 박물관, 갤러리, 방문자센터, 기념품거리 | 금문교 조망 최적화 |
| 해변·수변 구역 | 해변공원, 수상레저, 해안산책로, 서핑포인트 | 해안선 활용 극대화 |
| 생태공원 구역 | 자연보호구역, 식물원, 트레일, 야생동물관찰 | 녹지 보전 + 체험 |

### 외부 링 (반경 ~1.5km) — 생활·편의 중심

순환도로로 연결.

| 구역 | 시설 | 성격 |
|------|------|------|
| 주거 구역 | 에코하우스, 커뮤니티, 학교, 의료시설 | 자연 속 주거 |
| 상업 구역 | 로컬마켓, 레스토랑, 카페거리, 부티크 | 관광객+주민 공유 |
| 휴양 구역 | 리조트, 스파, 명상원, 글램핑, 요가파크 | 체류형 관광 |
| 교통 허브 | 순환도로, 트램, 자전거도로, 주차장 | 친환경 교통 |

### 해안축

해변에서 금문교까지 연결하는 산책로 + 뷰포인트 체인. 해안선 전체를 관광 동선으로 활용.

## 전문가 서브에이전트 상세

### 1. 정치 전문가

**분석 항목:**
- 토지 용도별 구역(zone) 경계 설정
- 환경보호 vs 개발 규제 프레임워크
- 공공·민간 거버넌스 모델
- 관광지 운영 정책

**산출물:**
- 구역별 토지이용 규제안
- 개발 허가 체계도
- 공공기관 배치 제안

**시네마틱 추천:** 공공건축물의 상징적 외관

### 2. 경제 전문가

**분석 항목:**
- 관광 수익 모델 (입장료, 체험, 숙박)
- 상업시설 유형·규모·배치 최적화
- 부동산 가치 구역별 분석
- 인프라 투자 우선순위

**산출물:**
- 상업시설 배치안 + 예상 규모
- 관광 동선별 수익 포인트
- 투자 단계별 로드맵

**시네마틱 추천:** 활기 있는 상업거리 장면

### 3. 사회 전문가

**분석 항목:**
- 주거 유형 (에코하우스, 커뮤니티)
- 편의시설 접근성 (의료, 교육, 복지)
- 문화·여가 공간 설계
- 주민-관광객 동선 분리

**산출물:**
- 주거 커뮤니티 설계안
- 편의시설 목록 + 위치
- 문화시설 프로그램

**시네마틱 추천:** 커뮤니티 생활 장면

### 4. 과학 전문가

**분석 항목:**
- 해안·산림 생태계 보전 전략
- 재생에너지 (태양광, 풍력, 조력)
- 지형·기후 조건 활용
- 해변 자연환경 보호 + 관광 양립

**산출물:**
- 생태보전 구역 지정안
- 에너지 자급 계획
- 해변 환경관리 가이드

**시네마틱 추천:** 자연과 건축의 조화 장면

### 5. 기술 전문가

**분석 항목:**
- 방사형 순환도로 + 보행자 도로망
- 친환경 교통 (트램, 자전거, EV)
- 스마트시티 센서·조명·안내
- 유틸리티 네트워크 (수도, 전력, 통신)

**산출물:**
- 도로망 설계도
- 교통 시스템 계획
- 스마트 인프라 배치안

**시네마틱 추천:** 트램·자전거 등 교통 장면

## 통합 조정 에이전트

### 역할

5개 전문가 보고서를 수합하여 다음을 수행:

1. **충돌 해소**: 구역 중복, 용도 상충, 보전 vs 개발 갈등 조율
2. **관광 최적화**: 시네마틱 뷰포인트 선정, 관광 동선 설계
3. **최종 문서 작성**: 종합 마스터플랜 + 구역별 상세 설계서

### 충돌 해소 기준 (우선순위 순)

1. 시네마틱 관광 가치 (최우선)
2. 생태 보전
3. 주민 생활 편의
4. 경제적 효율
5. 기술적 실현 가능성

## 시네마틱 관광 하이라이트

### 7개 장면 시퀀스

| Scene | 장면 | 설명 |
|-------|------|------|
| 1 | 금문교 전경 — 해변 일출 | 해변 수면 위로 금문교 실루엣 + 생태도시 스카이라인 |
| 2 | 다리 위 비행 — 도시 진입 | 금문교 케이블 사이로 비행, 방사형 도시가 펼쳐지는 순간 |
| 3 | 생태공원 저공비행 | 나무 캐노피 위를 스치며 트레일과 식물원 조감 |
| 4 | 해변 스위프 | 파도·서퍼·해변공원·수상레저를 따라가는 긴 테이크 |
| 5 | 주거 커뮤니티 전경 | 녹색 지붕 주택들, 태양광 패널, 커뮤니티 가든 |
| 6 | 상업거리 활기 | 보행자 거리의 노천카페, 로컬마켓, 거리 공연 |
| 7 | 금문교 석양 피날레 | 도시 전체를 내려다보며 석양에 물드는 금문교와 해변 |

### 핵심 경관 요소

**자연 요소:**
- 해안 절벽과 파도
- 안개가 금문교를 감싸는 장면
- 숲속 트레일의 빛 줄기
- 해변 석양 반사
- 야생 조류 군무

**인공 요소:**
- 녹색 지붕 건축물의 곡선
- 유리 전망대에서 비치는 하늘
- 보행자 다리 위 행인들
- 야간 조명 (따뜻한 톤)
- 트램이 달리는 순환도로
