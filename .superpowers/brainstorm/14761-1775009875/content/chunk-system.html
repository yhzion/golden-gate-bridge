Chunk System & City Generation

8km 반경 샌프란시스코를 브라우저에서 실시간 렌더링하는 핵심 전략

Chunk Grid — 카메라 중심 동적 로딩

City Generation Strategy

Building Types (절차적)

Highrise — 30~250m, 유리 커튼월, 다운타운
MidRise — 10~30m, 콘크리트/벽돌, 상업지역
Victorian — 8~15m, 베이윈도우, 주거지역
Industrial — 5~15m, 창고형, 항만/SoMa
Wharf — 5~10m, 목재 구조, 피셔맨스워프
각 타입은 BuildingFactory의 파라미터로 생성.
같은 타입 건물은 InstancedMesh로 병합 → 1 draw call.

Zone Map (구역별 건물 분포)

Financial District — Highrise 밀집
Marina / Pacific Heights — Victorian 밀집
SoMa / Mission — MidRise + Industrial 혼합
Fisherman's Wharf — Wharf + 관광시설
Sunset / Richmond — 저층 주택 반복
구역 정보는 좌표 기반 함수로 결정.
현재 terrainH()와 같은 방식의 zoneAt(x,z) 함수.

Chunk Lifecycle

1. IDLE → 카메라가 접근하면 → 2. LOADING
   • 지형 메시 생성 (즉시, CPU)
   • 건물 배치 계산 (시드 기반 결정론적)
   • InstancedMesh에 행렬 추가
2. LOADING → 완료 시 → 3. ACTIVE
   • LOD 레벨에 따라 디테일 조정
   • LOD 0: 개별 건물 형상, 창문 텍스처, 가로등
   • LOD 1: 건물 실루엣만, 텍스처 단순화
   • LOD 2: 높이만 반영한 단색 박스
3. ACTIVE → 카메라가 멀어지면 → 4. UNLOADING
   • 메시 dispose(), 행렬 풀에서 제거
   • 히스테리시스: 로드 거리보다 20% 더 먼 곳에서 언로드 (깜빡임 방지)
