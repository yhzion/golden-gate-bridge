Camera & Cinematic + Extensibility

시네마틱 카메라 시스템 + 지속적 확장을 위한 플러그인 아키텍처

Camera System — 3 Modes

CameraEffects — 속도/고도 반응형

속도 기반 효과
• Motion Blur: speed > 30m/s → 강도 비례
• FOV 변화: boost 시 60° → 75° (속도감)
• Chromatic Aberration: 초고속 시 미세하게
고도 기반 효과
• DoF: 저고도(< 50m) 에서 얕은 심도
• Fog falloff: 고고도에서 안개 위로 올라감
• Color temp: 고고도에서 차가운 톤
Extensibility — 지속적 확장을 위한 Registry 패턴

Cinematic Path — JSON 포맷

// src/camera/paths/golden-gate-sunrise.json
{
  "name": "Golden Gate Sunrise",
  "duration": 45,  // seconds
  "timeOfDay": "06:15",  // 자동으로 시간 설정
  "weather": "fog",
  "points": [
    { "pos": [-800, 50, -500], "lookAt": [0, 90, 640], "fov": 55 },
    { "pos": [-200, 80, 0], "lookAt": [0, 120, 640] },
    { "pos": [0, 72, 640], "lookAt": [0, 227, 1280], "fov": 40 },
    { "pos": [100, 250, 1400], "lookAt": [0, 80, 640], "fov": 65 }
  ]
}
새 시네마틱 경로 추가 = JSON 파일 하나 작성. CinematicDirector가 자동 로드.

Final Tech Stack

Build
Vite 6 + TypeScript 5.x
Rendering
Three.js r170+ (ES modules)
PostFX
Three.js EffectComposer + custom shaders
ECS (traffic)
miniplex (경량, TS 네이티브) or 자체 구현

현재 fbm 유지 + simplex-noise 패키지
State

UI
Vanilla DOM (현재 방식 유지)
Linting
